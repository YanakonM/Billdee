มี 2 ส่วน: (A) เก็บโค้ดบน GitHub และ (B) ให้แอปที่ติดตั้งแล้ว "กดตรวจหาอัปเดต → อัปเดตเอง"

## A. โค้ดอยู่บน GitHub แล้ว

Repo: **https://github.com/YanakonM/Billdee** — โค้ดทั้งหมด (รวม Billdee + Tauri) push ขึ้นแล้ว
อัปเดตโค้ดครั้งต่อไป: `git add -A && git commit -m "..." && git push`

## B. ระบบอัปเดตในแอป (ทำครั้งเดียว แล้วใช้ได้ตลอด)

### หลักการ
1. คุณ "ออกเวอร์ชันใหม่" โดย push tag (เช่น `v0.1.1`) ขึ้น GitHub
2. **GitHub Actions** build ตัวติดตั้งใหม่ + เซ็นชื่อ + สร้าง Release + ไฟล์ `latest.json` ให้อัตโนมัติ
3. ผู้ใช้เปิด Billdee → ตั้งค่า → ที่เก็บข้อมูล → **"ตรวจหาอัปเดต"** → **"อัปเดตเดี๋ยวนี้"** → แอปดาวน์โหลด ติดตั้ง รีสตาร์ทเอง (ข้อมูลไม่หาย)

### ตั้งค่าครั้งเดียว — เพิ่ม 2 Secrets บน GitHub

1. ไปที่ repo → **Settings → Secrets and variables → Actions → New repository secret**
2. เพิ่ม secret ชื่อ **`TAURI_SIGNING_PRIVATE_KEY`**
   - ค่า = เนื้อหาทั้งหมดในไฟล์ **`C:\Users\yanak\.tauri\billdee-updater.key`**
   - เปิดไฟล์ด้วย Notepad → คัดลอกทั้งหมด → วางเป็นค่า secret
3. เพิ่ม secret ชื่อ **`TAURI_SIGNING_PRIVATE_KEY_PASSWORD`**
   - ค่า = **เว้นว่าง** (กุญแจนี้ไม่มีรหัสผ่าน) — ใส่ค่าว่างแล้วกด Add

> 🔐 **สำคัญมาก:** ไฟล์ `C:\Users\yanak\.tauri\billdee-updater.key` คือกุญแจลับที่ใช้เซ็นอัปเดต
> **สำรองไฟล์นี้ไว้ที่ปลอดภัย** (เช่น USB ส่วนตัว) — ถ้าหาย จะออกอัปเดตให้แอปเดิมไม่ได้อีก
> และ **ห้าม commit ขึ้น GitHub** (.gitignore กันไว้แล้ว)

### ทำให้แอปที่ติดตั้งอยู่ "รองรับอัปเดต" (ครั้งเดียว)

แอปตัวที่ติดตั้งไปก่อนหน้านี้ยัง**ไม่มี**ระบบอัปเดต (เพิ่งใส่เข้าไป) ต้องติดตั้งตัวใหม่ทับ 1 ครั้ง:
- **วิธีง่าย:** `npm run tauri:build` แล้วติดตั้ง `Billdee_..._x64-setup.exe` ตัวใหม่ทับ
- จากนั้นเวอร์ชันถัดๆ ไปจะกดอัปเดตในแอปได้เลย ไม่ต้องลงเองอีก

---

## ออกเวอร์ชันใหม่ (ทุกครั้งที่จะปล่อยอัปเดต)

1. แก้โค้ดที่ต้องการ
2. **เพิ่มเลขเวอร์ชัน** ในไฟล์ `src-tauri/tauri.conf.json` บรรทัด `"version"`
   เช่น `0.1.0` → `0.1.1` (ต้องมากกว่าเดิม ไม่งั้นแอปจะไม่เห็นว่ามีอัปเดต)
3. Commit + push + ตั้ง tag ให้ตรงเวอร์ชัน:
   ```powershell
   git add -A
   git commit -m "v0.1.1"
   git push
   git tag v0.1.1
   git push origin v0.1.1
   ```
4. ไปดูที่ repo → แท็บ **Actions** จะเห็น "Release Billdee" กำลังทำงาน (ใช้เวลา ~5-10 นาที)
5. เสร็จแล้วจะมี Release ใหม่พร้อมไฟล์ติดตั้ง + `latest.json`
6. ผู้ใช้ทุกเครื่องกด **ตรวจหาอัปเดต** ในแอป → เห็นเวอร์ชันใหม่ → กดอัปเดต ✅
